Pretext 範例教學

10 個由淺入深的互動範例，每個都可以獨立執行

1. 量高度
prepare() + layout()
最基本的用法：算出一段文字在某個寬度下的行數和高度
2. Resize 響應
prepare() 一次 + layout() 多次
拖動滑桿改變寬度，觀察 layout() 即時重算（不需要重新 prepare）
3. 取得每行文字
prepareWithSegments() + layoutWithLines()
拿到每一行的文字和寬度，畫到 Canvas 上
4. Shrinkwrap 聊天氣泡
walkLineRanges()
找出不增加行數的最窄容器寬度，讓氣泡剛好包住文字
5. 文繞圖
layoutNextLine()
逐行排版，每行可以指定不同寬度，實現文字繞過圖片
6. Pre-wrap 模式
prepare(text, font, { whiteSpace: 'pre-wrap' })
保留空白、Tab、換行，適合 textarea 和程式碼編輯器
7. 切換語言與快取
setLocale() + clearCache()
切換分詞 locale 觀察斷行差異，手動清除快取釋放記憶體
8. Font 格式與注意事項
prepare(text, font)
正確的 font 格式範例、不要用 system-ui、lineHeight 要同步
9. 圓形文字排版
layoutNextLine() + 圓弦寬公式
每行用圓的弦寬當 maxWidth，文字自動排成圓形，可拖動滑桿調整半徑
10. 文字填滿自訂圖案
layoutNextLine() + 像素掃描
把大字畫到隱藏 Canvas 掃描像素，用小字填滿圖案輪廓（預設 SOS，可自訂）
