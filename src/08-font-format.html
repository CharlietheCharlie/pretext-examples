範例 8：Font 格式與注意事項

prepare() 的 font 參數格式跟 Canvas ctx.font 一樣（CSS font shorthand）。 以下展示各種正確和錯誤的格式，並用不同字型比較排版結果。

格式 範例 說明
size family 16px Inter 最基本的格式
weight size family bold 16px Inter 加粗體
style size family italic 16px Georgia 斜體
style weight size family italic bold 20px Arial 斜粗體
size "family name" 16px "Helvetica Neue" 家族名有空格要加引號
size family, fallback 14px "Courier New", monospace 多個 fallback
family size Inter 16px 順序錯了
size system-ui 16px system-ui macOS 上 Canvas/DOM 可能解析到不同字型
